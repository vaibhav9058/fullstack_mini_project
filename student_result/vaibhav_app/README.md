# Student Result Management System

A complete React CRUD application for managing student results with an attractive, modern dashboard UI.

## Features

- Add, View, Edit, Delete student records
- Search and filter functionality
- Sort by name, marks, or section
- Pagination support
- Form validation
- Statistics dashboard
- Responsive design

## Quick Start

1. Extract the ZIP file
2. Open terminal and run: npm install
3. In one terminal: npm run server
4. In another terminal: npm run dev
5. Open http://localhost:5173

## Tech Stack

- React 18
- JSON Server
- Vite
- CSS3 with gradients

Enjoy managing your student results!
